# 功能概览

MyObj 提供了丰富的文件管理功能，本文档将为你介绍主要功能特性。

## 文件上传与下载

### 智能上传

- **大文件分片上传**：支持 GB 级别文件上传
- **断点续传**：网络中断自动恢复
- **秒传功能**：基于 BLAKE3 哈希去重
- **多文件批量上传**：支持同时上传多个文件
- **文件加密存储**：可选择性加密敏感文件

### 灵活下载

- **多文件批量打包下载**：支持选择多个文件打包下载
- **离线下载**：支持 HTTP/HTTPS 资源下载
- **种子下载**：支持磁力链和 Torrent 文件下载
- **分享链接下载**：通过分享链接下载文件

## 文件管理

- **虚拟目录结构**：灵活的文件组织方式
- **文件操作**：重命名、移动、复制、删除（回收站机制）
- **搜索功能**：快速搜索文件和文件夹
- **文件预览**：支持图片、视频在线预览
- **自动缩略图**：为图片和视频自动生成预览缩略图
- **公开文件广场**：用户可以将文件设为公开，供其他用户浏览

## 用户与权限管理

- **用户存储配额**：为每个用户设置独立的存储空间限制
- **细粒度权限控制**：支持文件级别的访问权限设置
- **用户组管理**：基于用户组的权限策略控制
- **API Key 认证**：支持程序化访问和第三方集成
- **JWT 安全认证**：基于 Token 的现代化认证机制

## 文件分享

- **限时分享链接**：生成带有效期的文件分享链接
- **密码保护**：分享链接可设置密码
- **访问统计**：查看分享链接的访问次数

## WebDAV 支持

- **标准 WebDAV 协议**：兼容主流 WebDAV 客户端
- **文件浏览与下载**：通过 WebDAV 访问所有文件
- **目录管理**：创建、删除、重命名文件夹
- **权限控制**：基于用户权限的访问控制
