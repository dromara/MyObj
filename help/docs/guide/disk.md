# 挂载磁盘
初次使用系统需要管理员挂载磁盘，后续数据将存入该磁盘中，磁盘可以挂载多个，系统将根据文件大小自动选择合适的磁盘进行存储

# 挂载方式
点击`系统管理 - 磁盘管理 - 添加磁盘`  
可选择手动输入路径或扫描选择路径，系统扫描将无法修改磁盘大小
# 修改磁盘
点击`系统管理 - 磁盘管理`  
在磁盘列表中选中需要修改的磁盘，即可修改或删除。

# **注意**  
- 在系统正常使用中，不建议删除正在使用的磁盘，删除磁盘可能导致用户文件无法访问等问题！
- docker部署的用户在挂载磁盘后，需要将数据路径映射到本机磁盘，否则容器重启后可能造成数据丢失的问题。
- 磁盘空间不足时，用户上传文件将会失败。
- 因秒传功能存在，同一文件原则上不会在磁盘存在多个空间占用，所以，在所有用户均删除文件后，磁盘空间才会释放。
- 授权用户的空间大小可超过单个磁盘空间大小。
