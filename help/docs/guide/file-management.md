# 文件管理

## 文件上传

### 普通上传

1. 点击"上传"按钮
2. 选择要上传的文件
3. 选择保存位置
4. 点击"确定"开始上传

### 大文件上传

超过 1GB 的文件会自动启用分片上传：
- 支持断点续传
- 显示上传进度
- 网络中断后可自动恢复

### 秒传

如果服务器已存在相同文件，系统会自动识别并秒传，无需重新上传。

## 文件下载

### 单文件下载

点击文件列表中的"下载"按钮即可下载。

### 批量下载

1. 选择多个文件
2. 点击"批量下载"
3. 系统会自动打包下载

## 文件操作

- **重命名**：右键文件 → 重命名
- **移动**：右键文件 → 移动到
- **复制**：右键文件 → 复制到
- **删除**：右键文件 → 删除（会进入回收站）

## 文件搜索

在文件列表顶部的搜索框输入关键词，即可快速搜索文件和文件夹。

## 文件预览

支持以下文件类型的在线预览：
- 图片：JPG、PNG、GIF、WebP 等
- 视频：MP4、AVI、MOV 等
- 文档：PDF、TXT 等
