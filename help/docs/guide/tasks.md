# 任务中心

任务中心统一管理所有上传和下载任务，方便跟踪文件传输进度。

## 📤 上传任务

### 任务列表

在"任务中心"页面可以查看所有上传任务：
- **等待中** - 任务已创建，等待开始上传
- **上传中** - 正在上传文件
- **已完成** - 上传完成
- **已失败** - 上传失败
- **已取消** - 任务已取消

### 任务信息

每个任务显示：
- 文件名
- 文件大小
- 上传进度（百分比）
- 上传速度
- 已上传大小
- 剩余时间
- 任务状态

### 任务操作

- **暂停** - 暂停上传任务
- **继续** - 继续已暂停的任务
- **取消** - 取消上传任务
- **删除** - 删除任务记录

### 过期任务

系统会自动清理过期的未完成上传任务：
- 查看过期任务列表
- 续期过期任务
- 清理过期任务

## 📥 下载任务

### 任务类型

下载任务分为两类：

1. **离线下载任务**（type 0-6）
   - HTTP/HTTPS 下载
   - 磁力链接下载
   - 种子文件下载

2. **网盘文件下载**（type 7）
   - 从网盘下载文件到本地

### 任务列表

查看所有下载任务：
- **等待中** - 任务已创建，等待开始下载
- **下载中** - 正在下载文件
- **已完成** - 下载完成
- **已暂停** - 任务已暂停
- **已失败** - 下载失败
- **已取消** - 任务已取消

### 任务信息

每个任务显示：
- 文件名
- 文件大小
- 下载进度（百分比）
- 下载速度
- 已下载大小
- 剩余时间
- 任务状态
- 错误信息（如果失败）

### 任务操作

- **暂停** - 暂停下载任务
- **继续** - 继续已暂停的任务
- **取消** - 取消下载任务
- **删除** - 删除任务记录

## 🔄 任务管理

### 批量操作

- **一键清空** - 清空所有已完成/失败/已取消的任务
- **清空上传任务** - 清空所有上传任务
- **清空下载任务** - 清空所有下载任务

### 任务筛选

- 按状态筛选（全部、进行中、已完成、已失败）
- 按类型筛选（上传、离线下载、网盘下载）

### 任务统计

显示任务统计信息：
- 总任务数
- 进行中任务数
- 已完成任务数
- 失败任务数

## ⚡ 性能优化

### 断点续传

- 上传任务支持断点续传
- 下载任务支持断点续传
- 网络中断后自动恢复

### 并发控制

- 系统自动管理并发任务数
- 避免过多任务同时运行
- 优化系统资源使用

## 📊 任务详情

点击任务可以查看详细信息：
- 任务创建时间
- 任务更新时间
- 文件保存路径
- 错误信息（如果有）
- 任务日志

## 🔍 故障排查

### 任务一直等待

**可能原因**：
1. 系统资源不足
2. 网络连接问题
3. 服务器负载过高

**解决方案**：
- 检查系统资源使用情况
- 检查网络连接
- 等待系统资源释放

### 任务失败

**可能原因**：
1. 网络中断
2. 服务器错误
3. 文件不存在（离线下载）
4. 磁盘空间不足

**解决方案**：
- 查看错误信息
- 检查网络连接
- 检查磁盘空间
- 重试任务

### 任务速度慢

**可能原因**：
1. 网络带宽限制
2. 服务器性能限制
3. 文件过大

**解决方案**：
- 检查网络带宽
- 检查服务器性能
- 考虑分时段下载
