## 创建分享链接

1. 在文件列表中选择要分享的文件
2. 点击"分享"按钮
3. 设置分享参数：
   - **有效期**：设置链接有效期（小时）
   - **密码**：可选，设置访问密码
4. 点击"创建"生成分享链接

## 分享链接格式

```
http://your-domain/share/{share_id}
```

## 访问分享链接

### 无密码分享

直接访问分享链接即可下载文件。

### 有密码分享

1. 访问分享链接
2. 输入密码
3. 点击"验证"后下载文件

## 管理分享

在"我的分享"页面可以：
- 查看所有分享链接
- 查看访问统计
- 删除分享链接
- 修改分享设置

## 分享设置

- **有效期**：分享链接的有效时间
- **密码保护**：是否需要密码才能访问
- **下载次数限制**：限制下载次数（可选）
