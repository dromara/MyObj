# 用户权限

## 用户管理

### 创建用户

管理员可以在"系统管理" → "用户管理"中创建新用户。

### 用户组

系统支持用户组管理，可以为用户组分配不同的权限。

### 存储配额

可以为每个用户设置独立的存储空间限制。

## 权限控制

### 文件权限

- **私有**：仅用户自己可见
- **公开**：所有用户可见（文件广场）
- **分享**：通过分享链接访问

### 操作权限

- **上传**：上传文件权限
- **下载**：下载文件权限
- **删除**：删除文件权限
- **分享**：创建分享链接权限

## API Key

用户可以在设置页面创建 API Key，用于程序化访问。

### 创建 API Key

1. 进入"设置" → "API Key"
2. 点击"创建 API Key"
3. 设置名称和权限
4. 保存生成的 Key（只显示一次）

### 使用 API Key

在 API 请求头中添加：

```
X-API-Key: <your-api-key>
```
